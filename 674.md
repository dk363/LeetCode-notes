# 动态规划
dp[i] 代表包括当前元素的最长连续递增序列
```cpp
if (nums[i] > nums[i - 1]) {
    dp[i] = dp[i - 1] + 1;
} else {
    continue
    // dp[i] = 0;
}
```

```cpp
class Solution {
public:
    int findLengthOfLCIS(vector<int>& nums) {
        if (nums.empty()) return 0;
        int n = nums.size();
        
        // dp[i]表示以nums[i]结尾的最长连续递增子序列长度
        vector<int> dp(n, 1);  // 初始每个元素自身都是长度为1的子序列
        int max_len = 1;       // 记录全局最大值
        
        for (int i = 1; i < n; ++i) {
            // 状态转移：如果当前元素大于前一个，连续长度加1
            if (nums[i] > nums[i-1]) {
                dp[i] = dp[i-1] + 1;
            }
            // 否则保持初始值1（无需显式处理）
            
            // 更新全局最大值
            max_len = max(max_len, dp[i]);
        }
        
        return max_len;
    }
};
```
