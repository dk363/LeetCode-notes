# 滑动窗口
为什么会想到用滑动窗口呢？
左右指针的移动是有条件的 目的又是要找到左右之间的距离 距离可以通过左右指针相减得到
这里用到了一个小技巧 1 - nums 来结合 判定和加和
```cpp
class Solution {
public:
    int longestSubarray(vector<int>& nums) {
        int ans = 0, cnt0 = 0, left = 0;
        for (int right = 0; right < nums.size(); right++) {
            cnt0 += 1 - nums[right];
            while (cnt0 > 1) {
                cnt0 -= 1 - nums[left];
                left++;
            }
            ans = max(right - left + 1, ans);
        }
        return ans - 1;
    }
};
```
