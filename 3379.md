# 循环数组取到非负下标
通过一次取余去除超出 n 的部分
加上 n
通过第二次取余得到正数下标
```cpp
class Solution {
public:
    vector<int> constructTransformedArray(vector<int>& nums) {
        int n = nums.size();
        vector<int> res(n, 0);
        for (int i = 0; i < n; i++) {
            res[i] = nums[((i + nums[i]) % n + n) % n];
        }
        return res;
    }
};
```
