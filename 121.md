# 贪心
枚举price 在枚举的同时先更新最大的利润 然后更新最小的买入价格
这里就是要找到一个最大差的首位元素
在找到最小的买入价格之后 继续遍历
```c++
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int max_profit = 0, min_price = prices[0];
        for (int p : prices) {
            max_profit = max(max_profit, p - min_price);
            min_price = min(min_price, p);
        }
        return max_profit;
    }
};
```
