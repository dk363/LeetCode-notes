# 非法数据的判定
分情况讨论
对于左边：如果左边没有 nums[i - k] || (i > k && nums[i] > nums[i - k])
对于右边：如果右边没有 nums[i + k] || (i + k < n && nums[i] > nums[i + k])
那么这里就可以说这是一个好数
两种情况乘在一起 就有 4 种情况
那么这里非法条件的判断就非常重要了
```cpp
class Solution {
public:
    int sumOfGoodNumbers(vector<int>& nums, int k) {
        int goodSum = 0;
        int n = nums.size();
        for (int i = 0; i < n; i++) {
            // 这里先判断 是否合法
            // 如果合法 那么再判断 nums[i] 是否是最大的
            if ((i < k || nums[i] > nums[i - k]) &&
                (i + k >= n || nums[i] > nums[i + k])) {
                    goodSum += nums[i];
                }
        }
        return goodSum;
    }
};
```
