# 贪心 
最大的摆动数列
题目说明可以删除元素，意思就是遇到不合格的元素任然可以继续计数直到下一个合格的地方
这里一个难点就是遇到相等的元素如何计数

```c++
class Solution {
public:
    int wiggleMaxLength(vector<int>& nums) {
        int n = nums.size();
        if (n < 2) {
            return n;
        }
        int preDiff = nums[1] - nums[0];
        int ans = (preDiff == 0) ? 1 : 2;
        for (int i = 2; i < n; i++) {
            int currDiff = nums[i] - nums[i - 1];
            if ((preDiff <= 0 && currDiff > 0) || preDiff >= 0 && currDiff < 0) {
                ans++;
                preDiff = currDiff;
            }
            // 如果放在外面，就是每一次都更新preDiff
            // 如果这个时候是 0那么就覆盖掉之前的数据了
        }
        return ans;
    }
};
```
