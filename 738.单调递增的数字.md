# 贪心
对于原数字中没有单调递增的部分我们将最高位减一，后面全部改成 9
对于已经单调递增或者平 的部分，不需要管
```c++
class Solution {
public:
    int monotoneIncreasingDigits(int n) {
        string strNum = to_string(n);
        int l = strNum.size();
        int flag = l;
        for (int i = l - 1; i > 0; i--) {
            if (strNum[i - 1] > strNum[i]) {
                flag = i;
                strNum[i - 1]--;
            }
        }
        for (int i = flag; i < l; i++) {
            strNum[i] = '9';
        }
        return stoi(strNum);
    }
};
```
