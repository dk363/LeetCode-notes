# 二分查找
利用 matrix 的有序性 将 matrix 拼起来用
但是在代码实现时 用一种特殊的技巧来代替
对于拼起来的序列号为 i
## recall
### 闭区间 [left, right]
```cpp
left = 0, right = n - 1;
while (left <= right) {
    int mid = left + (right - left) / 2;
    if (nums[mid] < target) {
        left = mid + 1;
    } else {
        right = mid - 1;
    }
}
```
退出时 left == right + 1

### 左闭右开 [left, right)
```cpp
left = 0, right = n;
while (left < right) {
    int mid = left + (right - left) / 2;
    if (nums[mid] < target) {
        left = mid + 1;
    } else {
        right = mid;
    }
}
```
退出时 left == right

## 解法
```cpp
class Solution {
public:
    bool searchMatrix(vector<vector<int>>& matrix, int target) {
        int n = matrix.size(), m = matrix[0].size();

        int left = 0, right = m * n - 1;
        while (left < right) {
            int mid = left + (right - left) / 2;
            int num = matrix[mid / m][mid % m];
            if (target == num) return true;
            if (num < target) {
                left = mid;
            } else { 
                // target < num
                right = mid;
            }
        }

        return false;
    }
};
```
