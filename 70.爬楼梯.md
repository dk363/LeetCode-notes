# 递推 + 记忆化
对于 n > 1 这个问题可以拆分为更小的问题
拆分为 n - 1 n - 2
这个问题用递归的方法会造成超时
因此这里用地推的方法，并且通过引用记忆化之前已经计算过的数值
```c++
class Solution {
    vector<int> memo;
    int dfs(int i) {
        if (i <= 1) {
            return 1;
        }
        int& res = memo[i];
        if (res) {
            return res;
        }
        // 记忆化
        return res = dfs(i - 1) + dfs(i - 2);
    }
public:
    int climbStairs(int n) {
        memo.resize(n + 1);
        return dfs(n);
    }
};
```
