# 模拟
偶数的定义是能否被 2 整除
那么我们将问题简化：什么时候判断是否是偶数最简单？ 当这个数是 1 或者 2
那么将数字整除至100以内 数字位数最多为 2 此时判断最简单
```cpp
class Solution {
public:
    int findNumbers(vector<int>& nums) {
        int cnt = 0;
        for (int n : nums) {
            while (n >= 100) {
                n /= 100;
            }
            if (n >= 10) {
                cnt++;
            }
            // cnt += n >= 10;
        }
        return cnt;
    }
};
```
